ToggleButton

open class ToggleButton

Class that uses the Button class to emulate a toggle button.

Toggle buttons have a state, which represents whether they have been toggled on or off.

You use a toggle button in a manner similar to that of Button. See Button for usage instructions.

Bear in mind that this class is not a subclass of Button.

If using this class, do not call Button#update() and ToggleButton#update() in the same OpMode#loop() method. ToggleButton#update() calls Button#update() for you. Doing so yourself will result in undefined behavior.
