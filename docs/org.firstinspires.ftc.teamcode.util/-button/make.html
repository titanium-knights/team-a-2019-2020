open static fun make(gamepad: Gamepad!, buttonName: String!): Button!

Makes a button using a gamepad and a button name.

This method obtains the field automatically, so most op modes will use this method to create instances of Button.

Parameters

gamepad - Gamepad!: gamepad to track.

buttonName - String!: name of the button to track. Must be the name of a boolean property of Gamepad.

Exceptions

IllegalArgumentException - if the given button name does not exist on the gamepad.

Return
Button!: a gamepad that tracks a button on a gamepad.
