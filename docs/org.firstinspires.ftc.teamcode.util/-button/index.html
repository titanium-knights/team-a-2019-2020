Button

open class Button

Class that tracks a button on the gamepad and determines if it has been bumped (pressed then released).

Used to emulate push buttons in the FTC SDK.

Basic Usage

To use this class, create a Button instance using Button#make(Gamepad, String) during the OpMode#init() method. Pass Button#make(Gamepad, String) a gamepad (OpMode#gamepad1 or OpMode#gamepad2) and the name of a button on the gamepad (such as "dpad_left" or "x".)

Then, at the beginning of the OpMode#loop() method, call Button#update()..

Finally, to determine whether the button has been pushed/bumped, use Button#wasPressed() or Button#wasReleased().

See org.firstinspires.ftc.teamcode.TestOpMode for an example of how Button can be used.
