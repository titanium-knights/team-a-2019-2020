update

open fun update(): Unit

Updates the button's state, allowing it to track changes over time.

Call this method towards the beginning (or end) of an op mode's OpMode#loop() method.

Exceptions

RuntimeException - if the field does not exist on the gamepad. You should only see this if have directly invoked the constructor.
