FoundationClamps

class FoundationClamps

Helper class representing our robot's foundation clamps, which grip onto the foundation and pull it across the field.

This class uses an arbitrary number of CRServos to grip the foundation; each one can move in a different direction. Together, these servos extend and hook beams onto the ridges of the foundation.
