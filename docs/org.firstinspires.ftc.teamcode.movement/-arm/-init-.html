Arm(horizontal: DcMotor!, vertical: DcMotor!, horizontalMultiplier: Double, verticalMultiplier: Double)

Constructs a new arm object with a specified horizontal motor, vertical motor, horizontal multiplier, and vertical multiplier.

Parameters

horizontal - DcMotor!: motor that rotates the arm horizontally.

vertical - DcMotor!: motor that rotates the arm vertically (four-bar lift).

horizontalMultiplier - Double: multiplier for all powers sent to the horizontal rotation motor.

verticalMultiplier - Double: multiplier for all powers sent to the vertical rotation motor.
