standard

open static fun standard(hardwareMap: HardwareMap!): Arm!

Returns an Arm object preconfigured for our robot.

Specifically, it:

  • Retrieves and assigns the correct motors
  • Sets the Arm#horizontalMultiplier to -1, such that Arm#setHorizontalPower(double) moves the arm right when passed a positive power
  • Sets the Arm#verticalMultiplier to 1, such that Arm#setVerticalPower(double) moves the arm up when passed a positive power

Parameters

hardwareMap - HardwareMap!: hardware map of the robot.

Return
Arm!: an Arm object preconfigured for our robot.
